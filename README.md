# The Blinking Guy 👀

A tiny desktop app that reminds you to blink. Because developers forget.

It sits in your system tray and periodically flashes a pair of animated eyes on your screen — a gentle nudge to give your eyes a break.

## Features

- **Multiple eye styles** — classic, anime, pixel, minimal, and more
- **Configurable position** — pick any corner or centre of your screen
- **Adjustable timing** — set the reminder interval and how long the eyes stay visible
- **Click-through overlay** — the eyes never steal focus or block your work
- **System tray app** — runs quietly in the background

## Getting Started

```bash
npm install
npm start
```

Requires [Node.js](https://nodejs.org/) (v18+) and installs Electron automatically.

## Usage

- **Right-click** the tray icon → **Settings** to configure
- **Double-click** the tray icon to open settings
- Choose your preferred eye style, screen corner, interval, and display duration
